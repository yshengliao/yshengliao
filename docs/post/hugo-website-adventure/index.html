Now let’s dive into a few key commands and features that’ll make your Hugo journey a breeze (or at least less of a hurricane).

Important Hugo Commands and Tricks

  1. Create a new post: Need a new blog post? Easy peasy, just type:

    hugo new post/your-awesome-title.md

  2. Create your custom post template: Want to save time on formatting every new post? Create a custom archetype in the archetypes/ directory. Something like this will do:

    ---
title: "{{ replace .Name "-" " " | title }}"
date: {{ .Date }}
draft: true
author: "Sheng"
description: ""
tags: []
keywords: []
---

  3. Use your custom template to create a new post: Now that you’ve got your fancy template, you can use this command to create a new post based on it:

    hugo new --kind ../posts post/test.md

And there you have it! These are the essential tricks that’ll make working with Hugo feel less like a chore and more like you’re bending the internet to your will. Happy building!
